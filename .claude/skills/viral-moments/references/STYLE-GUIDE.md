# 病毒式文案风格指南

> **来源**: 基于真实案例分析
> **更新时间**: 2025-03-26
> **用途**: viral-moments skill 参考文档

## 核心特征

### 1. 短段落节奏

每段 1-3 句话，视觉上有呼吸感。

```
✅ 好的节奏：

当所有人都在担心 AI 会让人变懒

我发现了一个相反的现象

AI 淘汰的不是懒人

而是那些

不愿意思考的人

❌ 不好的节奏：

当所有人都在担心 AI 会让人变懒的时候，我发现了一个相反的现象，其实 AI 淘汰的并不是那些表面上的懒人，而是那些内心不愿意思考的人，这些人总是想着依赖工具...
```

### 2. 反常识观点

不随大流，制造认知冲突。

| 主流观点 | 反常识观点 |
|---------|-----------|
| AI 让人变懒 | AI 淘汰的是不愿思考的人 |
| 上四休三是福利 | 上四休三可能是效率陷阱 |
| 新技术要紧跟 | 不要对新技术那么热衷 |

### 3. 深度洞察

从现象提炼本质，引用原理/案例。

```
案例 1（杰文斯悖论）：

谁也逃不过一个

用效率提升来不断加码的机制

杰文斯悖论

你、我皆如此

---

案例 2（宗教祛魅）：

同时我对宗教这件事也彻底的祛魅

宗教是一种管理手段，仅此而已

不要去玷污信仰这个词
```

### 4. 鲜明立场

个人态度明确，不怕争议。

```
我不做任何资源整合性生意

我必须自己打通获客，产品，交付全流程

哪怕很多人告诉我

我错失了很多的机会

我以后会后悔
```

### 5. 有力结尾

短句收尾，留有余味。

```
怎么解？无解

边体会、边迭代、边继续往前冲

---

区别不在工具

在于你把工具当主人还是当奴隶
```

## 结构模板

### 模板 1：深度思考型

```
当所有人都在[热点/现象]

我发现了一个相反的事实

[支撑观察]

[深度洞察/原理]

[结尾留白]
```

**案例应用**：
```
当所有人都在担心 AI 会让人变懒

我发现了一个相反的现象

AI 淘汰的不是懒人

而是那些不愿意思考的人

区别不在工具

在于你把工具当主人还是当奴隶
```

### 模板 2：反常识型

```
[热点事件]引发了巨大震动

很多人在问：[主流疑问]

舆论哗然，大家都在说[主流观点]

但我并不这么认为

[反常识观点]

[原理/案例支撑]

[结尾]
```

**案例应用**：
```
张雪峰推行"上四休三"

他的离去引发了巨大震动

很多人在问：为什么

舆论哗然，大家都在骂"资本太黑"

但我并不这么认为

谁也逃不过一个

用效率提升来不断加码的机制

杰文斯悖论

你、我皆如此
```

### 模板 3：自我反思型

```
这周我[行动/观察]

结果很意外

[发现1]

[发现2]

不是[表面原因]

是[深层原因]
```

**案例应用**：
```
这周我刻意减少了一半的 AI 使用

结果很意外

我的思考反而更清晰了

不是 AI 不好

是我太容易把"思考"这个动作外包出去了
```

### 模板 4：问答型

```
学员问：[问题]

我的回答：

第一，[观点1]

第二，[观点2]

第三，[观点3]

[总结]
```

**案例应用**：
```
学员问：我不明白为什么很多明明是 AI 写的，流量就很高？

我的回答：

第一，幸存者偏差

第二，流量高不代表有变现

第三，就算有变现，你本身可能也挣不来这个钱

流量和变现是两码事
```

## 语言技巧

### 1. 制造转折

- "但我发现..."
- "但我并不这么认为"
- "相反的是..."
- "结果很意外"

### 2. 使用数字和排比

```
第一，幸存者偏差

第二，流量高不代表有变现

第三，就算有变现，你本身可能也挣不来这个钱
```

### 3. 对比增强说服力

```
不是 AI 不好

是我太容易把"思考"这个动作外包出去了
```

### 4. 留白结尾

```
怎么解？无解

边体会、边迭代、边继续往前冲
```

## 避坑指南

| ❌ 不要做 | ✅ 应该做 |
|---------|---------|
| 长篇大论 | 短段落，1-3句话一段 |
| 说教味浓 | 个人感悟，平等对话 |
| 无立场 | 观点鲜明，不怕争议 |
| 说得太尽 | 留白，让读者思考 |
| 陈词滥调 | 反常识，制造冲突 |
| 情绪煽动 | 理性分析，深度洞察 |
