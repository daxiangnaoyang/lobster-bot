# 病毒式朋友圈文案创作师 (Viral Moments)

一个帮助你创作具有传播力的社交媒体文案的 Claude Code Skill。

## 功能特点

- ✅ **短段落节奏**：自动生成 1-3 句话一段的呼吸感文案
- ✅ **反常识观点**：帮你找到不随大流的独特切入点
- ✅ **深度洞察**：从现象提炼本质，引用原理支撑
- ✅ **鲜明立场**：个人态度明确，不怕争议
- ✅ **多风格支持**：深度思考、自我反思、反常识、问答型

## 安装

### 方法 1：直接使用

将 skill 目录放在 `.claude/skills/viral-moments/` 下即可。

### 方法 2：打包安装

```bash
# 打包 skill
cd .claude/skills
python -m shutil.make_archive("viral-moments", "zip", "viral-moments")

# 重命名为 .skill 文件
mv viral-moments.zip viral-moments.skill
```

然后在 Claude Code 中安装。

## 使用方法

### 基础用法

```
请帮我写一个朋友圈文案，主题：XXX
```

### 指定风格

```
帮我写一个朋友圈文案，主题：AI 让人变懒，风格：反常识型
```

### 提供立场

```
热点：Sora 宣布全部关闭
我的观点：不要对新技术那么热衷
请帮我写朋友圈文案
```

### 优化现有文案

```
帮我把这段文案改成病毒式风格：
[你的文案]
```

## 支持的风格

| 风格 | 说明 | 适用场景 |
|------|------|---------|
| `deep` | 深度思考型 | 热点评论、行业洞察 |
| `reflection` | 自我反思型 | 个人成长、经验总结 |
| `contrarian` | 反常识观点型 | 制造争议、打破认知 |
| `qa` | 问答型 | 答疑解惑、观点输出 |

## 文案结构

### 标准模板

```
1. 引子（热点/观察/提问）
   ↓
2. 反常识观点（"但我并不这么认为"）
   ↓
3. 个人经历/观察支撑
   ↓
4. 深度洞察（引用/原理）
   ↓
5. 行动启示/留白结尾
```

## 示例输出

**输入**：
```
帮我写朋友圈文案，主题：AI 让人变懒，风格：反常识型
```

**输出**：
```
当所有人都在担心 AI 会让人变懒

我发现了一个相反的现象

AI 淘汰的不是懒人

而是那些

不愿意思考

只想当"工具人"的人

你越是依赖 AI 替你思考

你的思考能力退化得越快

但如果你把 AI 当作思考的杠杆

你会变得前所未有的强大

区别不在工具

在于你把工具当主人还是当奴隶
```

## 技巧要点

- ✅ 每段 1-3 句话，不超过 3 行
- ✅ 用"但我发现"、"相反的是"制造转折
- ✅ 数字、对比、排比增强说服力
- ✅ 结尾留白，不要说尽
- ❌ 避免长篇大论、说教味、无立场
