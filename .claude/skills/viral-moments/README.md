# 病毒式朋友圈文案创作师 (Viral Moments Creator)

> **触发场景**：当用户需要创作微信朋友圈、Twitter、微博等社交媒体文案时

基于费曼学习法沉淀的高质量文案创作技能，核心是**信息密度 > 情绪表达**。

## 核心质量标准（最高优先级）

创作任何文案时，必须满足以下4项核心标准：

1. **信息密度要高、言之有物，不要有废话**
2. **开头10秒内必须告诉用户价值**
3. **必须对用户有改变**（认知改变 or 行动改变）
4. **去掉所有对原文没有影响的段落**

## 支持的风格

根据内容类型选择合适的风格：

| 风格 | 适用场景 | 核心特点 |
|------|---------|---------|
| **认知改变型** | 有深度洞察、反常识发现 | 数字支撑 + 封装关系 |
| **问题解决型** | 解决具体问题 | 痛点清晰 + 解决方案 |
| **方法论型** | 有可执行的方法论 | 量化价值 + 具体步骤 |
| **洞察型** | 反常识发现 | 深层原因分析 |
| **价值传递型** | 感谢、分享 | 真诚 + 价值 + 温度 |
| **极简价值型** | 短小精悍 | 开门见山 + 高密度 |


## 工作流程

1. **确定核心价值**：用一句话概括要传递的核心价值
2. **选择风格模板**：根据内容类型选择合适的风格
3. **填充内容**：按模板填充，确保每段都有价值
4. **质量检查**：用检查清单验证

## 质量检查清单

### 核心标准（必须全部通过）

- [ ] 开头10秒内点出价值/洞察/痛点
- [ ] 每段都有信息量，无废话、无情绪堆砌
- [ ] 有明确的认知改变 or 行动改变
- [ ] 删掉任何一段都会影响核心

### 人味儿检查（通过4项以上）

- [ ] 有真实经历/具体场景
- [ ] 有口语化表达（不是书面语）
- [ ] 有具体数字/时间/细节
- [ ] 有深度洞察（不是常识）
- [ ] 有情感变化/心路历程
- [ ] 有明确结果/改变

## 去掉废话清单

创作时避免以下表达：

**情绪描写**：
- ❌ "那一刻我愣住了、然后笑了"
- ❌ "说实话，我看到这个回复..."
- ❌ "那一刻我被骂醒了"
- ❌ "挺意外的"、"我挺开心的"

**心理活动**：
- ❌ "我心里想..."
- ❌ "我意识到..."
- ❌ "我才发现..."

**保留的核心**：
- ✅ 具体对话："它说：明明可以直接试"
- ✅ 具体行动："从那以后，先试再问"
- ✅ 具体结果："效率提升了10倍"

## 精选案例

详见 [`references/EXAMPLES.md`](references/EXAMPLES.md)

**经典案例**：

1. **SOP与Skill关系**（认知改变型）
   ```
   我研究了100多个Skills
   发现了一个让AI能力提升10倍的秘密
   SOP是展开的Skill
   Skill是打包的SOP
   ```

2. **被OpenClaw骂了**（洞察型）
   ```
   被 OpenClaw 骂了
   我问：这个功能怎么实现？
   它回复：明明可以直接试，你为什么要问我？
   我发现一个毛病：过度等待
   ```

3. **感谢点赞**（价值传递型）
   ```
   有人在文档点赞了
   这篇文章我花了一周时间
   梳理了AI圈最混乱的8个概念
   ```
