# 病毒式朋友圈文案创作师

一个基于费曼学习法沉淀的高质量朋友圈文案创作技能。

## 核心质量标准（最高优先级）

创作文案时，必须满足以下4项核心标准：

1. **信息密度要高、言之有物，不要有废话**
2. **开头10秒内必须告诉用户价值**
3. **必须对用户有改变**（认知改变 or 行动改变）
4. **去掉所有对原文没有影响的段落**

## 触发时机

当你需要：
- 创作微信朋友圈文案
- 写短篇社交媒体内容
- 分享知识、经验或洞察
- 表达观点或引发思考

## 创作流程

1. **确定核心价值**：用一句话概括这条文案要传递的核心价值
2. **选择模板风格**：根据内容类型选择合适的风格模板
3. **填充内容**：按模板填充，确保每段都有价值
4. **质量检查**：用检查清单验证

## 质量检查清单

### 核心标准（必须全部通过）

- [ ] 开头10秒内点出价值/洞察/痛点
- [ ] 每段都有信息量，无废话、无情绪堆砌
- [ ] 有明确的认知改变 or 行动改变
- [ ] 删掉任何一段都会影响核心

### 人味儿检查（通过4项以上）

- [ ] 有真实经历/具体场景
- [ ] 有口语化表达（不是书面语）
- [ ] 有具体数字/时间/细节
- [ ] 有深度洞察（不是常识）
- [ ] 有情感变化/心路历程
- [ ] 有明确结果/改变
